# Config
- This folder is responsible for holding configuration files

## Server
- The server config file will allow the user to configure how the server works
  - I.E. how long to hold data files 
  - Change UI settings

## Client
- The client config file will allow the user to configure how the data collector works
  - I.E. what the IP address of the server is 
  - What the name of the client is
  - What the IP of the client is
